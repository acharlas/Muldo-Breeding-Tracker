Note — Endpoint Historique manquant
La vue Historique a besoin d'un endpoint GET /api/historique qui n'existe pas encore dans le backend (Phase 3 n'avait pas de route lecture des logs). Phase 4 devra soit ajouter cet endpoint au backend, soit utiliser les données de GET /api/cascade pour afficher le dernier cycle connu. Recommandation : ajouter GET /api/historique dans la même PR que le frontend.
